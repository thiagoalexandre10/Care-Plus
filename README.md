# CarePlus Challenge

Aplicação web desenvolvida para o Challenge CarePlus, com foco em melhorar a experiência do paciente no acompanhamento de consultas, notificações, gamificação e organização da rotina de saúde.

## Funcionalidades

- Login com CPF e senha
- Cadastro e recuperação de senha
- Home com próxima consulta, pontos, ranking e clima
- Calendário dinâmico
- Agendamento, reagendamento, cancelamento e confirmação de presença
- Chat assistente
- Sistema de pontos, conquistas e ranking
- Modo claro, escuro e personalização de cores
- Layout responsivo

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap
- Bootstrap Icons
- JavaScript
- LocalStorage
- API de clima

Reduzir faltas em consultas e melhorar a comunicação entre paciente e clínica, utilizando uma experiência digital mais clara, interativa e motivadora.
